# shared-config

# 프로젝트 소개

이 프로젝트는 자주 사용되는 ESLint 및 Prettier 또는 다른 설정 파일을 중앙에서 관리하여, 새로운 프로젝트를 설정할 때마다 반복적으로 설정 파일을 작성할 필요 없이 간편하게 가져다 사용할 수 있도록 돕습니다. 이를 통해 일관된 코드 스타일과 품질을 유지하고 (나의)설정 시간을 절약할 수 있습니다.
